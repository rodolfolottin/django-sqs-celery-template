## Getting started

### Requirements

Before running this app we will need:

- [Docker](https://www.docker.com/)
- [Docker Compose](https://docs.docker.com/compose/)

### Running locally (docker-compose)

1. Copy the .env.example

```
cp .env.example .env
```


2. Build and run the app:

```
docker-compose up
```
